Функция  isExists()  в FileStorage.java   выявлены дефекты:
  1. При передаче null в качестве аргумента функции происходит ошибка NullPointerException.

  2. Используемый метод contains проверяет, содержит ли String указанную последовательность символов. 
     В тестовом кейсе был создан файл с названием "TEST.txt".
     В связи с работой метода, обнаруживается совпадение с существующим файлом по комбинации символов, переданных в переменной,
     хранящей значение "TES". 
     В результате, ошибочно, функция возвращает true.
